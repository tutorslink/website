üìã Refund Policy Overview

Refunds from Tutors Link are only available within the first month of service.

Demo Class Refund

If a student and tutor attend a demo class and either party decides not to continue for any reason, the student is eligible for a 100% refund, provided no Tutors Link policies were violated (for example, sharing contact information during the demo class).

First Week Refund

If the student and tutor decide to continue after the demo class but later stop within the first week due to a valid issue (such as the tutor not attending sessions or the student behaving inappropriately), the student will receive:

  • A refund for all unattended classes
  • A refund for classes that had confirmed issues

Proof may be required. Tutors will still be paid for classes that were completed properly.
